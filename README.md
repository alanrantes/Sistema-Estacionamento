# Sistema de Estacionamento

![C#](https://img.shields.io/badge/C%23-239120?style=flat&logo=c-sharp&logoColor=white)
![Console App](https://img.shields.io/badge/Console_App-6DB33F?style=flat&logo=windows&logoColor=white)

Sistema de **estacionamento de veículos** desenvolvido em **C#**, que simula a gestão de um estacionamento, registrando a **entrada e saída de veículos** com armazenamento em arquivos **TXT**.

## Funcionalidades

- Registro da **entrada** de veículos  
- Registro da **saída** de veículos  
- Persistência de dados utilizando arquivos **TXT**

## Tecnologias

- C#  
- Arquivos TXT para armazenamento de dados  

## Como Executar

1. Clone o repositório:  
   ```bash
   git clone <URL_DO_REPOSITORIO>
2. Abra a solução no Visual Studio ou VS Code
3. Execute o projeto

## Objetivo do Projeto
Simular a gestão de um estacionamento, praticando conceitos de C#, persistência de dados e manipulação de arquivos.
